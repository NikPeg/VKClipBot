# VKClipBot

Бот для автоматической загрузки и публикации видеоклипов на стену группы ВКонтакте.

## Как работает

- Слушает сообщения в указанных беседах и скачивает прикреплённые видео через yt-dlp
- По расписанию публикует случайный клип из папки `clips/` на стену группы
- Периодически удаляет старые клипы

## Запуск через Docker

```bash
cp .env.example .env
# заполнить .env токенами и настройками

echo '{}' > state.json
docker compose up -d
```

## Запуск без Docker

```bash
pip install -r requirements.txt
# заполнить .env
python bot.py
```

## Конфигурация

Все настройки в `.env` (см. `.env.example`):

| Переменная | Описание |
|---|---|
| `BOT_TOKEN` | Токен группы |
| `USER_TOKEN` | Токен владельца |
| `GROUP_ID` | ID группы |
| `CONVERSATION_IDS` | ID бесед через запятую |
| `CLIP_DESCRIPTION` | Описание клипа при публикации |
| `START_HOUR` / `END_HOUR` | Окно постинга (UTC, т.е. МСК − 3) |
| `POSTING_INTERVAL` | Интервал между постами, часов |
| `CLEANUP_INTERVAL_HOURS` | Через сколько часов удалять старые клипы |

## Структура

```
clips/        # скачанные видео (не попадают в Docker-образ)
state.json    # время последнего поста (персистится между перезапусками)
.env          # секреты и настройки
```
